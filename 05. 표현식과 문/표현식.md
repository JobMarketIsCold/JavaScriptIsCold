# 표현식

표현식은 **값으로 평가될 수 있는 문**이다. 즉, 표현식이 평가되면 새로운 값을 생성하거나 기존 값을 참조한다.

리터럴은 값으로 평가되므로 리터럴도 표현식이다.

```jsx
// 리터럴 표현식
10;
("hello");

// 식별자 표현식
sum;
person.name;
arr[1];

// 연산자 표현식
10 + 20;
sum = 10;
sum !== 10;

// 함수/메서드 호출 표현식
square();
person.getName();
```

또한, 표현식은 값으로 평가되므로 문법적으로 값이 위치할 수 있는 자리에는 표현식도 위치할 수 있다.

```jsx
var x = 1 + 2;
x + 3; // 6
```
